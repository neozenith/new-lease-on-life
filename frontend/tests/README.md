# Playwright End-to-End Tests

To run Playwright tests:

1. Build and start your frontend (and backend if needed):
   ```sh
   npm run build
   npm run preview
   ```
   or for dev:
   ```sh
   npm run dev
   ```
2. In another terminal, run:
   ```sh
   npx playwright test
   ```

See [Playwright docs](https://playwright.dev/docs/test-intro) for more info.
